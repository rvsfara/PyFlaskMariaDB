Este Aplicativo busca demonstar o uso de Python, Flask e SQLAlchemy, 
consiste em um aplciativo web para efetuar um CRUD, DAO utilizando o mapeamento objeto relacional.

Flask é um framework leve que permite custruir um aplicativo web.

SQLAlchemy é um framework Python SQL para mapeamento relacional de objetos.

Flask-SQLAlchemy é uma extensão do FLask que adiciona suporte para o SQLAlchemy simplificando o desenvolvimento

Flask-wtf é uma extensão usada para criar componentes no formulário como rótulos, botões, campo de texto

Instalação
pip install Flask
pip install Flask-SQLAlchemy
pip install Flask-wtf
